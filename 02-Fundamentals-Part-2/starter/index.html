34. Function Declaration vs. Expression

  • Declaration: 
            function calcAge1(birthYear) {
          return 2037 - birthYear;
      }
      
      const age1 = calcAge1(1986);
  • Expression is function without a name: 
    const calcAge2 = function (birthYear) {
      return 2037 - birthYear;
  }
  
  const age2 = calcAge2(1986);
  • big difference: you can call declared function before they even have been declared

40. Basic Array Operations (Methods)

  • add Values to the end (with return value of the new length) with .push()
    const friends = ["Wurst", "Dennis", "Atze"];
    friends.push("Andi");
  • add Values on the start (with return value of the new length) .unshift()
    friend.unshift("Fist")
  • removes the last value and returns it .pop()
    friend.pop()
  • removes the first value and returns it .shift()
  • .indexOf() returns the first occurrence of a value: friends.indexOf("Dennis") result: 1. If not found in array result will be -1
  • .includes() detemines if an element is inside the array:
    friends.includes("Dennis") will be true.
    Uses strict equlity ===, so looking for "23" will not find 23

42. Introduction to Objects

  • structure: {key1: value1, key2: value2,...} (object literal syntax) 
    const stefan = {
      firstName: "Stefan",
      lastName: "Futterschneider",
      age: 2023 - 1986,
      job: "nice guy",
      friends: ["Wurst", "Dennis", "Atze"]
  };
  • in objects, compared to arrays the order doesn't matter, values are called via their keys
  • different ways to access the objects properties:
    stefan.lastname
    stefan["lastName"]     (bracket notation)
  • if using bracket notation, you can put expressions in the bracket:
    const nameKey = "Name";
    console.log(stefan["first" + nameKey]);
    console.log(stefan["last" + nameKey]);
  • add new properties works similar:
    stefan.location = "Deutschland";
    stefan["email"] = "fustef@mail.de"
  • also works nice in template literals:
    console.log(`${stefan.firstName} has ${stefan.friends.length} friends, and his best friend is ${stefan.friends[0]}.`);
    Output: Stefan has 3 friends, and his best friend is Wurst.
